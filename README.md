# carrotmarket_project
당근마켓 벤치마킹 프로젝트

#개발환경
- JDK 15.0.2
- STS 3.9.18, 
- ApacheTomcat9.0, 
- DB(Oracle),
- mybatis,
- spring framework(5.3.26) - Spring Legacy Project

#설계의 주안점
- 1주일씩 세분화한 기능을 각각 공부하여 만들어보고 제일 괜찮은 것을 채택할 예정 입니다.

#구현할 기능
- 로그인/로그인/회원가입)
- 기본적인 C, R, U, D 기능 구현
- 게시판(쇼핑몰 형식으로 이미지 게시판 구현 예정, 페이징 추가, 파일 업로드 추가, 이미지 리스트 가져와서 출력하기)
- 카카오맵 or 구글맵 지도 api 연동
- 채팅
